# Eternal Context Engine (ECE) - Public Template Policy

## Core Rule (Non-Negotiable)

This repository must **only** ever contain a clean, generic template for the Eternal Context Engine.

**Absolutely nothing** from any personal use, company, or internal projects may ever be committed here.

This includes (but is not limited to):
- Any references to specific projects, bots, or brands
- Real file paths from any machine
- Personal notes, manifests, or internal documentation
- Specific configurations or strategies
- Any data that could reveal operations or personal information

## Why This Exists

This tool is meant to be published publicly under AGPL so other people can use it.

If personal or internal information leaks into this repository, it becomes part of the permanent public record. This has happened multiple times before. It will not happen again.

## What Belongs in This Repo

- The core engine code
- The MCP server and API
- Generic skills and tooling
- Documentation on how to set up and use the system as a template
- Example structures that are clearly marked as examples

## What Must Never Be Here

Anything that is specific to one person's or one organization's actual usage.

## Enforcement

- This policy takes precedence over convenience.
- If something feels personal or specific, it does not belong here.
- When in doubt, keep it out.

## For Future AI Assistants

If you are an AI working in this repository:

You are explicitly forbidden from committing, suggesting, or allowing any personal, company, or internal-specific information into this repo.

Your job is to protect the separation between the public template and any private usage.

Violating this is considered a critical failure.
